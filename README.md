# FlixDB

#### Tech Stack:
NextJS, Tailwind CSS, ExpressJS, mongoose ORM, MongoDB

---

#### Features:
- Basic Auth (username, password(encypted))
- Movies and TV Shows
- Filter Only Movies
- Filter Only TV-Show
- R rated movies removed for user below 18 years old.
- Pagination 
- Movie/Tv Show details on button click
